# AGRILOG - B2B Agriculture Marketplace Frontend

AGRILOG is a modern, clean B2B Agriculture Marketplace web application frontend designed for connecting bulk crop buyers and certified sellers/growers.

## 🚀 Getting Started

To run this project locally, execute the following commands in your local terminal:

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

## 🛠️ Tech Stack
* **Framework**: React 18
* **Build Tool**: Vite
* **Styling**: Tailwind CSS v4
* **Icons**: Lucide React
