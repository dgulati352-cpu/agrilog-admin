# agrilog-admin

# AGRILOG Admin Panel & B2B Ecosystem Platform

Operational Admin Hub & B2B Agriculture Marketplace for AGRILOG ([`agrilog-pi.vercel.app`](https://agrilog-pi.vercel.app)).

## 🌾 Overview
The **AGRILOG Admin Panel** serves as the central operational hub for managing user verifications (Farmers, Equipment Partners, Domestic & Enterprise Buyers), crop demand data, machinery rental listings, buyer-farmer marketplace transactions, commission tracking, localized content delivery, and RBAC security.

## 🚀 Features
- **Module 1: User & KYC Verification Management** (Interactive document zoom/rotate viewer, status approval/rejection, verified badges)
- **Module 2: Crop Demand & Advisory Engine** (Market demand index portal, target tonnage, crop lifecycle guides)
- **Module 3: Equipment & Rental Management** (Machinery catalog moderation, rental bookings log, dispute resolution modal)
- **Module 4: Marketplace & Transaction Coordination** (Harvest listings moderation, 5-stage contract negotiation tracker, logistics delivery)
- **Module 5: Revenue & Commission Management** (Configurable commission rates, transaction & escrow release ledger, B2B SaaS subscriptions)
- **Module 6: Multilingual Content & Broadcast Engine (CMS)** (9-language localization dictionary & targeted push broadcast alert engine)
- **Module 7: Platform Analytics & Dashboard** (GTV, Net Revenue, MRR, 8-state regional supply vs demand heatmap)
- **Module 8: Audit Logs & System Config** (Immutable audit trail log, RBAC role scopes, PII masking privacy policy)

## 🛠️ Tech Stack
- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS v4 + Outfit/Inter Typography
- **Icons**: Lucide React
- **Backend / Database**: Firebase Auth & Firestore
